# Petful Laravel Project

## Dependences

Copy .env.example to .env file

```shell
cp .env.example .env
```

Install docker dependences

```shell
docker-i
```

Run docker containers

```shell
up
```

Install and update yarn and composer dependences

```shell
devdep
```

## Database

Run migrations

```shell
ms
```

## Running project

Run project

```shell
run
```

In wsl or production, build project

```shell
build
```
